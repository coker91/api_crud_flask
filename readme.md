## Packages Requirement

```sh
Flask==3.0.1
marshmallow==3.21.1
flask-marshmallow==1.2.1
Flask-SQLAlchemy==3.1.1
```
## To install the requirements package, run this command
```sh
pip install -r requirements.txt
```
## running project 
```sh
python3 run.py
```
